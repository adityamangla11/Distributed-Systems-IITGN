## Assignments

### 1. [Distributed Text Processing Service](./Distributed_File_Processing_Service/)

A containerized microservices application for distributed text processing operations using a hub-and-spoke architecture. A Central Server routes client requests to specialized worker services (Uppercase, Lowercase, Reverse, Word Count) running as Docker containers.

**Tech Stack:** Python · Flask · Docker · Docker Compose
